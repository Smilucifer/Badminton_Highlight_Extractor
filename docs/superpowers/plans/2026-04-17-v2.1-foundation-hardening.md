# v2.1 Foundation Hardening Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a configurable, observable, and failure-visible foundation for the badminton highlight pipeline without changing the core segmentation behavior.

**Architecture:** Introduce a single `AppConfig` object that owns all current hard-coded thresholds and runtime settings, then thread it through the main orchestration flow and the three pipeline stages. Add preflight validation, stage-level statistics, and explicit subprocess failure handling so later accuracy work can be debugged and compared reliably.

**Tech Stack:** Python 3, dataclasses, OpenCV, Ultralytics YOLO, librosa, SciPy, FFmpeg, argparse

- [ ] **Step 3: Create `config.py` with the default config dataclasses**

```python
from dataclasses import asdict, dataclass, field
from typing import Any


@dataclass(slots=True)
class VisionConfig:
    model_path: str = "yolov8n-pose.pt"
    batch_size: int = 16
    num_workers: int = 4
    sample_fps: float = 5.0
    confidence_threshold: float = 0.5


@dataclass(slots=True)
class AudioConfig:
    sample_rate: int | None = None
    hop_length: int = 512
    peak_height: float = 0.015
    min_peak_distance_sec: float = 0.2


@dataclass(slots=True)
class FusionConfig:
    motion_percentile: float = 50.0
    hit_window_before: float = 0.2
    hit_window_after: float = 1.2
    min_intense_ratio: float = 0.05
    rally_gap_threshold: float = 2.5
    min_rally_duration: float = 1.0


@dataclass(slots=True)
class ExportConfig:
    min_highlight_score: float = 80.0
    pad_front: float = 1.5
    pad_back: float = 2.0
    max_workers: int = 5


@dataclass(slots=True)
class AppConfig:
    vision: VisionConfig = field(default_factory=VisionConfig)
    audio: AudioConfig = field(default_factory=AudioConfig)
    fusion: FusionConfig = field(default_factory=FusionConfig)
    export: ExportConfig = field(default_factory=ExportConfig)

    def to_dict(self) -> dict[str, Any]:
        return asdict(self)


def get_default_config() -> AppConfig:
    return AppConfig()
```
